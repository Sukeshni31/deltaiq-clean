# DeltaAIQ — AI Readiness Benchmark

A lightweight tool that benchmarks any company's AI readiness by industry, generating an AIQ score with context and recommendations.

🔗 **Live:** [deltaiq-clean.vercel.app](https://deltaiq-clean.vercel.app)

---

## What it does

Enter a company name and select an industry. DeltaAIQ generates an **AIQ score** — a benchmarked view of where that company sits on AI readiness relative to its sector.

**Industries covered:**
- IT Services
- Aviation
- Manufacturing
- Pharma / Healthcare
- SaaS / Product
- GCC / Captive

---

## Why I built it

GCC and enterprise HR leaders increasingly need to assess AI readiness — for hiring decisions, capability planning, and workforce strategy. This tool gives a fast, structured starting point.
